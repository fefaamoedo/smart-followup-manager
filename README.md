# Smart Follow-Up Manager

![Dashboard Executivo](assets/dashboard.png)

Aplicação analítica desenvolvida em **Python + Streamlit** para gestão de pendências, atrasos e priorização operacional em processos de follow-up de compras e suprimentos.

---

## Visão Geral

Este projeto simula um cenário empresarial onde pedidos precisam ser acompanhados continuamente com base em:

- atraso de entrega  
- valor financeiro  
- criticidade do item  
- comprador responsável  
- status do follow-up  
- prioridade de atuação  

A proposta é transformar controles manuais em uma solução visual, estruturada e orientada à tomada de decisão.

---

## Objetivo de Negócio

Apoiar áreas de:

- Compras  
- Supply Chain  
- Planejamento  
- Operações  
- Suprimentos  

na identificação rápida de pendências prioritárias, reduzindo atrasos e melhorando o controle operacional.
